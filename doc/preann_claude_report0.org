* Project Overview

*PREANN* stands for *Parallel Reinforcement Evolutionary Artificial Neural Networks*. This is an academic research project that implements a framework for flexible multi-layer artificial neural networks with reinforcement learning based on genetic algorithms, featuring parallel implementations using SIMD instructions (SSE2/XMM) and NVIDIA's CUDA for GPU acceleration.

** Purpose

The project explores the intersection of:
- Neural networks (configurable multi-layer architectures)
- Genetic algorithms (evolutionary optimization)
- Reinforcement learning (task-based fitness evaluation)
- High-performance computing (parallel implementations)

** Current Status

According to the README, this project is marked to be "replaced by tetani, a library written in rust" (from the most recent commit in 2019 which says "Perhaps the library should move to rust").

* Code Organization

** Architecture Patterns

- *Factory Pattern:* The factory module creates objects with different implementations (C++, SSE2, CUDA) transparently.
- *Template Method Pattern:* Buffer and Connection classes define abstract interfaces implemented differently for each buffer type and implementation.
- *Strategy Pattern:* Different algorithms (selection, crossover, mutation) are interchangeable through enumeration-based selection.

* Interesting Features

** 1. Multi-Implementation Comparison Framework
The project can compile the same neural network with three different implementations (C++, SSE2, CUDA) and directly compare their performance. This is unique for its time (2009-2012).

** 2. Sophisticated Parameter Sweep System
The Loop framework allows nested iteration over parameters to systematically explore hyperparameter spaces, automatically generating plots and performance data.

** 3. Bit-Packed Neural Networks
Support for BT_BIT buffer type allows neural networks to use single bits instead of floats, dramatically reducing memory usage for binary networks.

** 4. Multiple CUDA Strategies
Four different CUDA calculation strategies (REDUC0, REDUC, OUT, INV) exploring different parallelization approaches for neural network computation.

** 5. Hand-Optimized Assembly
SSE2 assembly code manually optimized for SIMD operations on XMM registers - uncommon in neural network libraries even today.

** 6. Comprehensive Genetic Algorithm Implementation
Multiple crossover levels (weight, neuron, layer), multiple crossover algorithms (uniform, proportional, multipoint), and multiple selection strategies all configurable.

** 7. Board Game Integration
Includes a complete Reversi/Othello game implementation to test neural networks in game-playing scenarios.

** 8. Academic Rigor
Extensive benchmarking suite, performance analysis, and documentation reflecting academic research standards.

** 9. Early GPGPU Adoption
This project used CUDA for neural networks in 2009-2012, predating the deep learning revolution by several years. It shows early exploration of GPU acceleration for AI.

** 10. Modular Buffer Types
The abstraction of different data representations (float, bit, sign, byte) with automatic conversions is architecturally sophisticated.

* Summary

PREANN is a well-architected academic research project from 2009-2012 that explored parallel implementations of evolutionary neural networks. It successfully demonstrates GPU acceleration (CUDA) and SIMD optimization (SSE2) for genetic algorithm-based neural network training, achieving significant performance improvements.

The project was completed as a master's thesis and represents early pioneering work in GPGPU computing for AI, predating modern deep learning frameworks. The codebase is mature, well-tested, and documented, but has been inactive since 2019 with plans to migrate to Rust.
